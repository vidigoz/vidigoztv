Las herramientas del Hilado

Las tres herramientas con las que se teje el Tempoverso. Cada una debe estar corriendo en su puerto.

Inicia cada herramienta con node server.js desde su carpeta antes de abrir el enlace.
🎬
Vidiclip
El Clip
Editor de stories tipo 9:16. Texto RSVP sobre fondos de imagen o video, música, título y exportación a MP4. Integra con R2 y Facebook para publicación.
Abre directo — no necesita servidor aparte
📖
Vidiserial
El Lector
Lector RSVP vertical para crear videos tipo story. Slideshow de imágenes y videos, fuentes configurables, velocidad ajustable, exportación a MP4 1080×1920.

Vidiwrite
El Escribano
Editor de historias con sugerencias de IA en tiempo real. Marcadores [[descripción]] para que Claude complete fragmentos. Exporta a Notion.
